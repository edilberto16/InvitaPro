# InvitaPro v2.6.4 — Catálogo Global de Plantillas

## Qué cambia
El cliente ya no queda atrapado en la categoría elegida al crear la invitación.

Desde `Mi InvitaPro Studio → Cambiar plantilla` ahora puede navegar:

- Recomendadas
- Todas
- Bodas
- XV años
- Cumpleaños
- Empresarial

## Cambio entre categorías
Puede pasar, por ejemplo:
`Unicornio → Dinosaurios → Luxury Black → Modern Editorial`

Se conservan:
- Nombre y textos
- Fecha y hora
- Ubicación
- Portada y galería
- Música
- Itinerario
- Dress code
- Mesa de regalos
- RSVP

Solo cambia la capa visual de la plantilla.

Si aplica una plantilla diseñada originalmente para otra categoría, InvitaPro muestra una confirmación antes de cambiar.

## También incluye
- Categoría visible en cada tarjeta.
- Etiqueta Premium.
- Vista previa individual antes de aplicar.
- Indicador de plantilla actual.
- Filtros del catálogo.

## Instalación
1. Copia el PATCH sobre tu proyecto actual.
2. No requiere SQL.
3. Ejecuta:
   `pnpm run build`
4. Luego:
   `pnpm dev`
5. Prueba:
   `Mi InvitaPro → Continuar editando → Cambiar plantilla`
