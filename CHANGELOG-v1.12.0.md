# InvitaPro v1.12.0 — Catálogo completo de plantillas

## Cambios

- Se habilitaron todas las plantillas del catálogo.
- Cada diseño cuenta ahora con una identidad visual inicial propia.
- Se añadieron estilos para bodas, XV años, infantiles y eventos empresariales.
- Las plantillas continúan usando el mismo motor, contenido, RSVP, galería, música y ubicación.
- Se actualizó la versión del paquete a `1.12.0`.

## Plantillas habilitadas

Bodas: Elegant Classic, Luxury Black, Royal Gold, Minimal White, Romantic Garden, Sunset, Vintage y Modern Editorial.

XV años: Princess Rose, Golden Night, Butterfly, Lavender, Glamour, Floral y Luxury Pink.

Infantiles: Safari, Dinosaurios, Unicornio, Espacial y Power Heroes.

Empresariales: Corporativo, Lanzamiento, Conferencia y Networking.

## Base de datos

Esta actualización no requiere migraciones ni cambios en Supabase.
