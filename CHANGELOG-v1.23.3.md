# InvitaPro v1.23.3 — Limpieza de experiencia del editor

## Cambios

- Se eliminaron referencias técnicas visibles a `design_json`, Supabase Storage y la implementación de persistencia.
- Theme Studio ahora comunica la personalización con un texto orientado al usuario.
- El editor de propiedades explica de forma clara qué puede personalizarse.
- La ayuda para ordenar secciones ahora se limita a la acción de arrastrar o usar las flechas.
- El estado de la música seleccionada utiliza un mensaje simple y comercial.

## Compatibilidad

- No requiere cambios SQL.
- No modifica la estructura de datos existente.
- Es compatible con la Biblioteca Multimedia y el selector de recursos de v1.23.2.
