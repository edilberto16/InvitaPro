# InvitaPro v1.13.0 — Premium Template Engine

## Nuevo

- Registro central de identidad visual para todas las plantillas.
- Paletas independientes por plantilla: color principal, secundario, fondo, superficie, texto y texto tenue.
- Familias de diseño: bodas, XV años, infantil y empresarial.
- Modos de composición: classic, editorial, immersive, playful y business.
- Sistemas tipográficos: serif, editorial, rounded y modern.
- Decoraciones configurables: ornamental, botanical, sparkle, geometric y cosmic.
- Variables CSS compartidas para que las secciones existentes adopten automáticamente el tema seleccionado.
- Base preparada para ordenar y sustituir secciones en las siguientes versiones.

## Compatibilidad

- No requiere cambios de base de datos.
- Respeta `color_principal` como personalización del cliente.
- Las invitaciones existentes sin plantilla reconocida usan Elegant Classic como respaldo.
