# InvitaPro v2.6.2 — Personalización completa

Incluye:
- Portada desde Biblioteca Multimedia.
- Galería desde Biblioteca Multimedia (hasta 8 imágenes).
- Música desde Biblioteca Multimedia.
- Fecha y hora editables.
- Lugar, dirección y Google Maps editables.
- Itinerario, dress code, regalos y RSVP guardados.
- Vista previa móvil actualizada con portada, fecha y lugar.
- El cambio de plantilla ahora se refleja también en la vista previa pública usando `template_key`.
- Conserva todo el contenido al cambiar plantilla.

No requiere SQL nuevo.

## Instalación
1. Copia el PATCH sobre el proyecto actual.
2. `pnpm run build`
3. `pnpm dev`
4. Prueba:
   Mi InvitaPro → Continuar editando → Portada/Galería/Música/Ubicación → Guardar cambios → Vista previa.
