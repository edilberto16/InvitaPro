# InvitaPro v2.0.5 — Inspiración con identidad propia

## Cambios

- Cada experiencia de Inspiración tiene ahora una identidad visual independiente.
- Boda Luxury Gold: composición editorial dorada.
- Boda Romantic Garden: estilo botánico y orgánico.
- XV Princess Rose: composición rosa, juvenil y brillante.
- Cumpleaños Dino Adventure: tipografía infantil, colores vivos y formas dinámicas.
- Baby Shower Teddy Clouds: tonos suaves, nubes y formas redondeadas.
- Future Summit: diseño corporativo, geométrico y tecnológico.
- Las tarjetas del catálogo también reflejan la identidad de cada plantilla.
- Se mantiene una sola base funcional reutilizable para RSVP, cuenta regresiva, ubicación y CTA.
- Se eliminó del login el mensaje técnico relacionado con `.env.local`.

No requiere SQL, nuevas variables de entorno ni dependencias adicionales.
