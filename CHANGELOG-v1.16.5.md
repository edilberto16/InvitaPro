# InvitaPro v1.16.5

## Sistema visual administrativo reutilizable

- Se agregaron componentes compartidos para acciones, grupos de acciones, estados y pantallas vacías.
- Clientes, Eventos, Invitados y Confirmaciones ahora usan el mismo sistema visual.
- Se normalizaron tamaños, alineación, colores, estados deshabilitados y comportamiento responsive.
- Los estados negativos y borradores ahora tienen colores semánticos diferenciados.
- La mejora evita que futuros cambios de estilos se apliquen de forma distinta entre módulos.

No requiere cambios en la base de datos.
