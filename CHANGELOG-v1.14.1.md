# InvitaPro v1.14.1

## Vista previa administrativa para borradores

- El botón **Vista previa** en Invitaciones siempre abre con `?preview=1`.
- El Dashboard permite previsualizar invitaciones en borrador, pausadas o publicadas.
- Los pases personalizados también admiten vista previa administrativa en borrador.
- La vista previa valida que exista una sesión autenticada antes de consultar datos privados.
- El enlace público normal continúa bloqueando invitaciones que no estén publicadas.
- Se agrega un aviso visible para distinguir la vista administrativa del enlace público.

No requiere cambios SQL ni migraciones.
