# InvitaPro v1.11.1 — Selección directa de plantillas

## Cambios

- El botón **Usar plantilla** abre el constructor con el diseño elegido.
- La plantilla seleccionada se aplica automáticamente al formulario nuevo.
- Se conserva el color principal definido por cada diseño.
- El catálogo del constructor se abre en la colección correspondiente.
- Se mantienen deshabilitadas únicamente las plantillas que todavía están en desarrollo.
- No requiere cambios ni migraciones en la base de datos.

## Instalación

Copia el contenido de este parche sobre la carpeta principal del proyecto y reemplaza los archivos existentes.
